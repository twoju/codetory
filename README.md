## 💡 프로젝트 개요
디자인 학부 졸업 프로젝트로 초등학생을 위한 앱 서비스를 기획, 디자인하고 오프라인 전시에서 사용자와 상호작용하기 위해 코드 기반 프로토타이핑 툴인 Protopie 를 활용해서 서비스를 제공한 경험이 있습니다. 개발을 학습한 후 **프로토타입이 아닌 실제 개발을 통해 MVP를 구현**하고자 하는 마음으로 시작했습니다.
